# Barracuda Trials Pathfinder

A RuneLite plugin for efficient Sailing skill training through Barracuda Trials. Uses A\* pathfinding to calculate optimal routes in real-time, navigating around hazards while maximizing speed boosts.

![Barracuda Trial Example](docs/barracuda_trial_example.png)

## Features

- **Real-time pathfinding** - Dynamic calculation that adapts to moving obstacles to path between waypoints
- **Static Waypoint routing** - Wiki-driven paths for optimal waypoint order
- **Dynamic highlighting** - Different colors for the objectives you need to be looking at
